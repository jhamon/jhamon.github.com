A jQuery/javascript impelementation of the classic arcade game. The game UI is entirely handled by jQuery DOM manipulations. A scoreboard is provided by a Rails backend. AJAX calls to the Rails JSON API allow for repeated play without any hard page refreshing. The source is available on github.
